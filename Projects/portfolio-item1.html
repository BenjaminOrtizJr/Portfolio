React Todo App

React App

This is a Todo App made using the JavaScript library React.

This a react app that is utilizing hooks to manage the state of the application. The app allows the user to create a list of tasks. All tasks listed have an edit and delete button that allow the user to edit and update the text or to remove the task from the list.

The app was deployed using Netlify.
