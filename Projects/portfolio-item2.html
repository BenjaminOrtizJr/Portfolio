@ Twitter Clone

React App

This is a Twitter clone made using React.

This SPA was inspired by Twitter and made using the JavaScript library React. The app utilizes all CRUD methods and allows the user to read, create, edit, and delete all posts. This project does not have a backend or database and utilzizes an array of objects to hold the loaded data. When a post is created a new object is created and added to the array of objects. The application state is being passed to components via prop drilling.

This app was deployed using Netlify.
