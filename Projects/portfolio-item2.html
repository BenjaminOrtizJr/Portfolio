C-3PO Profile App

Public API App

This is a public API app made using JavaScript.

This is a app built using HTML5, CSS, JavaScript, and the Star Wars public API. Axios is being used to get the information from the API. The app is retrieving all the information for the Star Wars character C-3PO. The characters personal traits are being displayed on left beside the image of C-3PO. The right side is displaying all the movies C-3PO starred in. All this information is coming from the public Stars API in JSON format and is being rendered by creating elements in Javascript and utilizing the DOM.
