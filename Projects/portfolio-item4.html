User Manager App

Full Stack App

Full Stack React App.

This is a an app that allows you to enter a user's information that is stored in a mySQL database and then the data is rendered in the app along with an edit and delete option for each entry.The front end was built in React and styled using Bootstrap.

This app was deployed using Netlify for the frontend and heroku for the backend. I am also hosting the MYSQL database on heroku using the Heroku add-on ClearDB MYSQL.
