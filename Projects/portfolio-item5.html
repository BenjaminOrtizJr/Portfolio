Live Chat App

Full Stack App

This is a live chat application made using React for the frontend and Nodejs, Express, and Socket.io for the backend.

This live chat app allows the user to start a live chat just by entering their name and room name. Users can join existing chat rooms by entering the same chat room name or create a new room by entering a different chat room name. Once the chat session has ended, all history of the chat is deleted. No registration or authentication is required. Allowing this to be a quick, easy, and anonymous chat experience.The front-end was built in React. The backend was built using Node and the Express web framework along with Socket.io for real time communication with web sockets.

The frontend was deployed using Netlify and the backend with Heroku.
